# Кулинарный форум-помощник

Локальное Django-приложение для поиска рецептов, подбора блюд по продуктам,
публикации пользовательских рецептов и обсуждения блюд.

## Возможности

- регистрация, вход и личный кабинет;
- каталог рецептов с поиском, фильтрацией и сортировкой;
- подбор рецептов по имеющимся продуктам;
- случайное блюдо из всей базы или по введенным продуктам;
- отправка пользовательских рецептов на модерацию;
- комментарии, оценки, избранное и жалобы;
- панель модерации для управления рецептами и комментариями;

## Запуск

```bash
py -m pip install -r requirements.txt
py manage.py migrate
py manage.py loaddata recipes/fixtures/demo_data.json
py manage.py runserver
```

Демо-администратор после загрузки фикстур:

- логин: `admin`
- пароль: `admin12345`

## Проверки

```bash
py manage.py test
py -m flake8 .
```
